# LMS Design System

## Style Direction
Soft Premium Learning UI.

The interface should feel clean, calm, professional, friendly and white-label ready.

## Color Tokens

- Primary: `#7C3AED`
- Primary Dark: `#5B21B6`
- Primary Soft: `#EDE9FE`
- Accent: `#F97316`
- Accent Dark: `#EA580C`
- Accent Soft: `#FFEDD5`
- Heading: `#17231B`
- Body: `#5F6472`
- Background: `#F8F7FC`
- Surface: `#FFFFFF`
- Border: `#E7E3F5`

## Typography

- Heading font: Playfair Display
- Body/UI font: Inter

## UI Rules

- Use rounded cards and soft shadows.
- Use purple for primary actions.
- Use orange for high-attention conversion buttons.
- Keep public pages light, friendly and easy to scan.
- Build every component so it can later be powered by database content.
