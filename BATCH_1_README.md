# LMS Batch 1 - Public Website Pages

This batch adds the public-facing frontend foundation for the white-label LMS.

## Files included

- `index.html`
- `pages/courses.html`
- `pages/course-details.html`
- `pages/about.html`
- `pages/pricing.html`
- `pages/contact.html`
- `assets/css/style.css`
- `assets/js/app.js`
- `docs/design-system.md`
- `docs/ai-build-log.md`

## How to apply

Copy the contents of this zip into the root of your local Git repo.

Then run:

```powershell
git status
git add .
git commit -m "Add public website pages"
git push -u origin feature/public-pages
```

If you are not already on the public pages branch, run this first:

```powershell
git checkout -b feature/public-pages
```
